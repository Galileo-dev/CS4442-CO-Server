# Java Multi-Threaded Server

## Description

A simple Java multithreaded client and server for two-way communication

## Usage

### Server

```
java Server <port>
```

## Build

```
mvn clean package
```

the executable `.jar` file will be in the `target/` directory

## Test

```
mvn test
```
